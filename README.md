## 制作背景

---

## 工夫した点

### 制作事例を最優先とした情報設計
採用担当者様が最も確認したい情報は“どのような制作物を作れるのか”だと考え、制作事例をファーストビュー直後に配置しました。
また、その後にスキル一覧やプロフィールを設置することで、技術力だけでなく人物像についても段階的に理解していただける情報設計を意識しています。

### 統一感と保守性を意識したコーディング
フォントサイズやカラーなどを変数で管理し、ユーティリティクラスも活用することで、サイト全体に統一感を持たせました。
また、モバイルファーストで実装を進めることで、スマートフォンからPCまでレイアウトが自然に変化するレスポンシブ対応をおこなっています。
制作後の修正や機能追加にも対応しやすいよう、保守性を意識したコーディングを心がけました。

### 人柄を反映したサイトデザイン
ポートフォリオサイトは制作物だけでなく、制作者自身の人柄や価値観も伝える場だと考えています。
そこで全体は誠実さや落ち着きを表現できるグレーブルーを基調とし、プロフィールページでは「人に寄り添う」という自身の価値観を表現するため、温かみのあるオレンジをアクセントカラーとして採用しました。
色彩設計を通して、技術面だけでなく人物像も伝わるデザインを目指しています。
